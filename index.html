📖 Comment jouer ?

Le but : isoler l'inconnue x en gardant la balance en équilibre, comme dans une vraie équation.

  • 🟥 Les carrés rouges sont les inconnues (x), leur masse est secrète.
  • 🔵 Les cercles bleus sont les constantes (des « 1 »).
  • ➕ Mode + (par défaut) : clique un poids dans la banque pour l'ajouter des deux côtés. Tu peux aussi agréger (glisser un poids sur un autre).
  • ➖ Mode − : le bouton − remplace la banque par une corbeille. Dépose un poids dedans pour le retirer ; deux identiques = retrait des deux côtés.
  • ➗ Pouvoir ÷ : ouvre une échelle ; glisse vers la droite pour choisir le diviseur (2, 3, 4…) — chaque poids se découpe en secteurs — puis relâche pour diviser les deux côtés. × arrive bientôt.
  • 🏦 La Banque (en haut à gauche) : clique un poids pour l'ajouter, puis glisse-le dans un bac.
  • 🧲 Glisse un poids sur un autre du même type pour les combiner (1 + 1 = 2…).
  • 🌧️ Un poids négatif glissé sur un positif les annihile (antimatière).
  • ✂️ Double-clique un poids pour le diviser.
  • 🔨 Le marteau découpe un poids pour faire apparaître la valeur présente sur l'autre plateau (sinon, découpe au hasard).
  • 🎉 Quand un côté ne contient plus que x et l'autre un nombre, c'est gagné !
